เกมดับเดนจักรวาล

เกี่ยวกับ

ย้อนกลับไปช่วงประมาณปี พ.ศ. 2540 (นานมากกก~) ผมได้มีโอกาสได้เล่นเกมนี้อยู่นิดหน่อยในเครื่องคอมพิวเตอร์ของโรงเรียน จากการลองค้นหาดูเหมือนเกมนี้จะสูญหายหาในอินเตอร์เน็ตไม่พบไปแล้ว ด้วยความที่ยังมีแผ่น CD เกมนี้อยู่ (จริงๆ คือแผ่นโปรแกรมดิกชันนารีที่มีตัวเกมนี้แถมมาให้ด้วย) และก็โชคดีมากๆ ที่แผ่นยังเปิดอ่านได้อยู่ เลยตัดสินนำเกมนี้มาเผยแพร่อีกครั้ง น่าจะเป็นเกมคอมพิวเตอร์เกมแรกๆ ที่เป็นภาษาไทย

เกมนี้เป็นเกมของสำนักงาน ป.ป.ส. จากร่องรอยที่พบในระบบไฟล์ของเกม คาดว่าเกมนี้น่าจะพัฒนาโดยคุณลาภลอย วานิชอังกูร (Laploy V.Angkul) และเมื่อฟังเสียงพากย์ภาษาไทยก็ชัดเจนว่าเป็นเสียงของน้าต๋อยเซมเบ้

"เป็นเกมต่อสู้กึ่งผจญภัย ที่สอดแทรกความรู้เกี่ยวกับโทษพิษภัยของยาเสพติด ผู้เล่นเกมจะได้รับบทเป็นอนิรุธ พระเอกของเรื่อง ซึ่งจะได้รับมอบหมายให้ไปปราบขุนทศ ราชายาเสพติด และต้องช่วยเหลือตัวประกันที่ถูกจับ 5 คน ในเกมจะประกอบด้วยด่านทั้งหมด 5 ด่าน ที่ผู้เล่นต้องเดินทางผ่าน โดยในแต่ละด่านจะมีเส้นทางวกวนเหมือนเขาวงกต ซึ่งหากผู้เล่นสามารถต่อสู้กับศัตรู และเก็บไอเทมต่างๆ รวมทั้งจดจำแคปซูลข้อมูล เพื่อใช้ในการตอบคำถามได้อย่างถูกต้อง ผู้เล่นก็จะสามารถช่วยเหลือตัวประกัน และผ่านเข้าสู่ด่านต่อไปได้"

ดาวน์โหลด

thcd.zip (37 MB) (165 Downloads) แตกไฟล์ zip ออกมาก่อน จะได้เป็นไฟล์ ISO Image ของตัวแผ่น CD

เท่าที่ทดสอบ เกมไม่สามารถเปิดเล่นได้บน Windows รุ่นใหม่ น่าจะเล่นได้เฉพาะ Windows 95/98/ME เท่านั้น (ผมลองทดสอบกับ Windows 98 SE แล้วเล่นได้) ใครอยากลองเล่นจริงๆ ก็แนะนำให้ใช้โปรแกรม VM อย่างเช่น VirtualBox หรือ VMWare แล้วติดตั้ง Windows 98 เอา (ขอไม่อธิบายนะครับ ให้ไปหาวิธีใช้โปรแกรม+วิธีติดตั้ง Windows 98 กันเอาเอง) ให้ทำการ mount ไฟล์ ISO เข้ากับตัว Windows 98 แล้วเข้าไปในแผ่น ไฟล์ตัวเกมจะชื่อว่า SOLAHERO.EXE (ในแผ่นจะมีโปรแกรมดิกชันนารีด้วย เป็นโปรแกรมสุดคลาสสิกอีกอันเหมือนกัน)
